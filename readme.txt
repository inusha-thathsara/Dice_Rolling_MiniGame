Node.js Dice Game
======================

A simple multiplayer dice game built using Node.js, Express, and Socket.IO.

Description
---------------

This program allows multiple players to join a game and roll a virtual dice. The player with the highest roll wins the round. The game continues until a player leaves or a new round is started.

Features
------------

* Multiplayer support using Socket.IO
* Real-time updates of player rolls and winners
* Simple and intuitive user interface
* Automatic detection of multiple players and enabling/disabling of the "Roll" button accordingly

Technologies Used
--------------------

* Node.js
* Express.js
* Socket.IO
* HTML/CSS for the user interface

How to Run
--------------

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `node game.js` to start the server
4. Open multiple tabs in a web browser and navigate to `http://localhost:3000/game.html` to play the game
